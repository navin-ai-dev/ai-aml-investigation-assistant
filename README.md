# 🏦 AI AML Investigation Assistant

## Overview

AI-powered Anti-Money Laundering (AML) Investigation Platform designed to assist compliance teams, investigators, and risk analysts in detecting suspicious transactions, generating investigation reports, managing AML cases, and monitoring SLA breaches.

## Key Features

### AI Investigation Engine

* Risk Scoring
* AML Pattern Detection
* Compliance Review
* Executive Summary Generation
* Suspicious Activity Report (SAR) Generation

### Case Management

* Case Creation & Tracking
* Investigator Assignment
* Investigation Notes
* Status Workflow Management
* Case Details View

### Operations Dashboard

* Manager Dashboard
* SLA Monitoring
* Escalation Dashboard
* Priority Analytics
* Risk Distribution Analytics

### Data Persistence

* SQLite Database
* Case History Storage
* Investigation Retrieval
